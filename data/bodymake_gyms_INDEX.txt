================================================================================
DATA FIELD SPECIFICATIONS
================================================================================

Each gym entry contains 14 structured fields:

1. name (string)
   - Gym brand name in Japanese/English
   - Examples: "BEYOND", "24/7Workout", "チキンジム"

2. prefecture_slug (string)
   - Regional code: tokyo, osaka, fukuoka, nagoya, yokohama, 
                   saitama, chiba, sendai, sapporo, hiroshima, 
                   kobe, kyoto

3. address (string)
   - Representative location(s), station names
   - Examples: "Shinjuku, Shibuya", "Tenjin, Hakata"

4. price_min (number, JPY)
   - Minimum 2-month package price
   - Range: ¥12,000 (パーソナルジムNEXT) to ¥115,000 (UNDEUX)

5. price_max (number, JPY)
   - Maximum package price (premium courses)
   - Range: ¥100,000 to ¥660,000 (RIZAP)

6. price_trial (number, JPY)
   - Trial/consultation fee: 0 = free, 11000 = BEYOND
   
7. price_enrollment (number, JPY)
   - One-time enrollment fee: 0 = free to 55000 (RIZAP)

8. trial_available (boolean)
   - true = trial session available
   - false = not offered

9. catchphrase (string)
   - Marketing tagline in Japanese
   - Examples: "結果にコミット" (RIZAP), "3食食べるダイエット" (24/7)

10. description (string)
    - 50-200 character description of gym philosophy/USP
    - Includes key facts: success rate, trainer qualifications, etc.

11. programs (array of strings)
    - Offered programs: ダイエット, ボディメイク, 短期集中, etc.

12. website_url (string)
    - Full HTTPS URL to gym's official website

13. options_wear (boolean)
    - Free wear/clothing rental available

14. options_shoes (boolean)
    - Free shoe rental available

15. options_protein (boolean)
    - Protein drink/supplement provided

16. options_diet (boolean)
    - Diet guidance/meal plan included
